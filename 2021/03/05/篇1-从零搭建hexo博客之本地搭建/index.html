前言

本人也算不上什么大神,主要就是看百度的同类教程写的很不详细.有很多问题都没有提出解决方法.而本篇教程会详细到每一个命令.

Heox将成为一个分类,这个分类里包含了从hexo的本地安装到配置文件的讲解,再到部署到远端(github pages/云服务器).希望这篇文章以及这个分类能够帮助到你.

  • 最后一步把launch git bash勾上,吧view release note取消.然后finish
  • 此时会出来一个窗口,类似于cmd命令提示符窗口.如图:↓
  • 在这个窗口中输入git --version

创建译并预览站点

创建站点

windows:

  • 在你所希望的位置新建一个文件夹,名字任意但必须是英文.并打开它.
  • 在如图位置输入cmd并回车,这会打开一个命令提示符窗口,输入以下命令 hexo init.
  • 如图,顺利的话,会输出INFO Start blogging with Hexo!这样的内容.
  • 然而,事实可能不那么顺利.那么你可以尝试在左下角的windows菜单中以图中的步骤打开命令提示符.然后cd进入你刚刚创建的文件夹.文件夹路径可以从上个步骤中输入cmd的地方找到,ctrl+c复制它!然后cd [path].然后再次执行hexo init.
  • 如果还不行的话我建议你在评论区求助.

    linux

  • 同样,在你所希望的位置新建一个文件夹,名字任意但必须是英文.并打开它.
  • 然后cd到这个目录,用linux就不用我教怎么cd了吧.执行sudo hexo init.
  • 这可能有点慢,没办法,这是看运气的,毕竟那么高一堵墙挡着.
