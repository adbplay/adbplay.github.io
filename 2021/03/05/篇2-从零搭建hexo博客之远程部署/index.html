前言

本篇文章观看不需要太高的计算机基础(Linux/unix除外),但前提是完全的按照文中步骤一步一步地走.

在阅读本篇文章之前请先阅读 篇1-从零搭建hexo博客之本地搭建

GIT平台

目前较为出名的(只要是我没听过的都不出名(:-))有三家,分别是:

  • github

  • gitee

  • coding

    其中GitHub知名度最高,是全球最大的同性交流额不是…..开源交流社区.
    gitee次之,coding最后.如果你对本排名有异议,那么你是对的,不用跟我争论.

而GitHub的服务器位于美国,在国内的访问速度一言难尽,而gitee和coding都是国内的.所以本文以国外github,国内gitee分别为例进行操作教程.

优/劣 势

  • github:自动部署,可以自定义域名 / 访问速度慢,每次push之后自定义域名会掉,需要重新设置
  • gitee:访问速度快 / 不可以自定义域名,每次push之后都要手动更新部署
  • coding:emmm…..没用过,跳过
